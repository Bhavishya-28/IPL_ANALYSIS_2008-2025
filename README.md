# 🏏 IPL Analysis (2008–2025) – Power BI

---

## 📌 Project Overview

This project focuses on analyzing Indian Premier League (IPL) data from 2008 to 2025 using Power BI.
The goal is to extract meaningful insights about team performance, player achievements, and match trends.

---

## 📊 Key Features

* Season-wise performance analysis
* Orange Cap & Purple Cap winners
* Top players based on fours and sixes
* Team performance comparison
* Interactive filters and slicers

---

## 🛠️ Tools & Technologies

* Power BI
* DAX
* SQL (for data preparation)
* Excel / CSV datasets
